# Prerequisites
- **C++ Compiler** (e.g., GCC)
- **Cnpy Library**
  
# Steps to build and install cnpy

 ```

        git clone https://github.com/rogersce/cnpy.git 
        cd cnpy
        mkdir build
        cd build
        cmake ..
        make
        sudo make install

 ```

# How to run and verify the C++ kernel 

Run the shell program

  ```
        ./run_all.sh
  ```
